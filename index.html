Privacy Policy

Last updated: June 2025

Spectra is a Chrome extension that audits HTTP security headers, cookies, and CORS configuration for websites you visit. This policy explains what data the extension accesses and how it is handled.

Data We Access

  • HTTP response headers — read from pages you navigate to, to evaluate security configuration. This data never leaves your device.
  • Cookies — read from the active tab to analyze cookie security attributes (HttpOnly, Secure, SameSite). This data never leaves your device.
  • Page URLs — used to fetch headers and perform security analysis. URLs are not stored persistently or transmitted to our servers.

External Requests

Spectra makes requests to the following third-party services:

  • hstspreload.org — to check whether a domain is on the browser HSTS preload list. Only the domain name is sent. No personal data is included.
  • Websites you choose to scan — when using the Bulk Scanner, the extension fetches HTTP headers directly from URLs you provide.

No data is sent to Spectra's servers. We do not operate any data collection infrastructure.

Local Storage

  • Session storage — scan results and header data are stored temporarily in chrome.storage.session and are automatically cleared when the browser is closed.
  • Local storage — your Pro license key is stored in chrome.storage.local on your device. It is never transmitted to third parties.

Data Sharing

We do not sell, share, or transmit any personal data to third parties. The extension does not collect analytics, crash reports, or usage statistics.

Permissions

Spectra requests the following Chrome permissions:

  • webRequest — to read HTTP response headers from pages you visit
  • cookies — to read cookie attributes for the active tab
  • tabs — to identify the current active tab URL
  • storage — to cache scan results locally and store your license key
